A fully fledged e-commerce admin module for Laravel 5.1+.

## Install

Via Composer

``` bash
$ composer require league/laravel-market
```

## Usage

``` php
$skeleton = new League\Skeleton();
echo $skeleton->echoPhrase('Hello, League!');
```
